PERMUTATIONS AND COMBINATIONS — JEE Mathematics

=== FUNDAMENTAL COUNTING PRINCIPLE ===
If event A can happen in m ways and event B in n ways:
AND (both): m × n ways
OR (either): m + n ways

=== PERMUTATIONS ===
Arrangements where order matters.
P(n, r) = nPr = n! / (n-r)!
= n × (n-1) × (n-2) × ... × (n-r+1)

All arrangements of n distinct objects: P(n, n) = n!
Arrangements with repetition: n^r ways for r selections from n objects

Permutations with repeated elements:
n objects with p identical, q identical, r identical:
= n! / (p! * q! * r!)

Circular permutations of n distinct objects = (n-1)!
Circular with necklace (no flip distinction) = (n-1)!/2

=== COMBINATIONS ===
Selections where order does NOT matter.
C(n, r) = nCr = n! / (r! * (n-r)!)

Properties:
nCr = nC(n-r)
nC0 = nCn = 1
nC1 = n
nCr + nC(r-1) = (n+1)Cr

=== SELECTION WITH REPETITION ===
r items from n types (repetition allowed) = C(n+r-1, r)

=== IMPORTANT RESULTS ===
Number of subsets of n-element set = 2^n
Number of non-empty subsets = 2^n - 1
Number of diagonals in n-gon = nC2 - n = n(n-3)/2

=== DERANGEMENTS ===
D(n) = n! * Σ (-1)^k / k!  for k from 0 to n
D(n) = n! * (1 - 1/1! + 1/2! - 1/3! + ... + (-1)^n/n!)
D(1)=0, D(2)=1, D(3)=2, D(4)=9

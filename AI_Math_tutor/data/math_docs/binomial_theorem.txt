BINOMIAL THEOREM — JEE Mathematics

=== BINOMIAL THEOREM ===
(a + b)^n = Σ_{r=0}^{n} C(n,r) * a^(n-r) * b^r

where C(n,r) = n! / (r! * (n-r)!)  (nCr)

=== GENERAL TERM ===
T_{r+1} = C(n,r) * a^(n-r) * b^r   (r = 0, 1, 2, ..., n)
Middle term: T_{(n/2)+1} if n is even; T_{(n+1)/2} and T_{(n+3)/2} if n is odd

=== USEFUL SPECIAL CASES ===
(1 + x)^n = Σ C(n,r) x^r
(1 - x)^n = Σ C(n,r) (-x)^r = Σ (-1)^r C(n,r) x^r

Sum of coefficients: put x=1 → 2^n
Sum of odd-position coefficients = Sum of even-position coefficients = 2^(n-1)

=== COMBINATORIAL IDENTITIES ===
C(n,r) = C(n,n-r)           (symmetry)
C(n,r) + C(n,r-1) = C(n+1,r)  (Pascal's identity)
Σ C(n,r) = 2^n              (sum of all)
C(n,1) = n
C(n,2) = n(n-1)/2
C(0,0) = C(n,0) = C(n,n) = 1

=== MULTINOMIAL THEOREM ===
(a + b + c)^n = Σ n!/(p!q!r!) * a^p * b^q * c^r
where p + q + r = n

=== APPLICATIONS ===
Finding specific terms: set exponent of x equal to required power
Coefficient of x^r in (1+x)^n = C(n,r)
Greatest coefficient: middle term (when n even) or both middle terms (when n odd)
Greatest binomial coefficient: C(n, n/2) for even n
